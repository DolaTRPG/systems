# 基礎數值

  • 屬性：
    • 其中三種 ability defense：12
    • 其餘三種 ability defense：11
  • 血量（hit points）：8
  • 探索速度：120 ft
  • 戰鬥速度：40 ft

# 起始裝備

每個角色起始會有

  • 兩天份的食物
  • 一套武裝（二選一）
    • 一個武器、一個防具、一個頭盔或盾牌
    • 一本法書（請洽主持人）
  • 冒險道具（總價值 200 銅以內）
